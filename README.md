# Airline Operations Analytics Dashboard

## Project Overview

This project analyzes U.S. airline on-time performance data to identify delay patterns by airline, airport, time of day, and delay cause.

The goal is to use Python, SQL, and Power BI to clean, analyze, and visualize flight operations data, then provide business recommendations that can help improve airline and airport performance.

## Business Questions

1. Which airlines have the highest and lowest delay rates?
2. Which airports experience the most delays?
3. What time of day has the highest risk of delays?
4. What are the main causes of delays?
5. How can airlines and ground operations teams reduce delay impact?

## Tools Used

- Python
- Pandas
- SQL
- Power BI
- GitHub

## Dataset

Data will come from the Bureau of Transportation Statistics On-Time Performance dataset.
